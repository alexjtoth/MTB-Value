# 🚵 MTB Market

MTB Market is a web application for estimating the fair market value of used mountain bikes.

The goal is to make buying and selling mountain bikes more transparent by combining bike specifications, original MSRP, historical pricing, and market data into one platform.

---

## Current Features

- 🔍 Search bikes by brand, model, category, and year
- 🎛 Dynamic filtering
- 📄 Individual bike detail pages
- 🗄 Supabase database
- ⚡ Built with Next.js and TypeScript

---

## Tech Stack

- Next.js 15
- React
- TypeScript
- Tailwind CSS
- Supabase
